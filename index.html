우리는 3개월간 데잇걸즈 과정을 수료하면서 매우 압축적으로 성장했습니다.

자연스럽게 그동안 배운 데이터 분석 스킬을 써먹고 싶었습니다.

실제 회사에서는 어떻게 쓰이는지, 앞으로 데이터 분석가가 된다면 어떤 일에 부딪히게 될지, 우리의 이전 경험과는 어떻게 연결될지 궁금했기 때문이죠.

메타데이터가 가진 강점 4가지를 무기삼아 사회의 문을 두드려본 경험을 여러분께 들려드리겠습니다.

데이터를 설계하고 예쁜 집을 지어드려요

비즈니스에서도 데이터 분석에서도 올바른 KPI를 설정하는 일은 무척 중요합니다.

비즈니스 목표를 달성하기 위한 핵심 지표를 정의하는 것부터 구글 애널리틱스 데이터 분석 환경 구축, 데이터 대시보드 설정까지 도와드립니다.

4륜안으로 데이터가 숨겨놓은 의미를 발굴합니다

메타데이터는 경영, 마케팅, 엔지니어링, 통계까지 4개 분야에서 평균 2년 이상의 경험을 가지고 있는 실무자가 모였습니다. 각 분야의 도메인 지식을 가지고 데이터의 의미를 보다 빠르게 핵심 지표를 찾아 낼 수 있습니다.

그래서요? 뭘 하면 되냐고요?

분석을 위한 분석, 테크니컬을 우선하는 분석보다는 실제 비즈니스에 도움이 되는 액션을 발굴합니다. 데이터를 보고 ‘아하! 그랬군요. 그럼 이렇게 해볼게요.’를 말할 수 있게 말이죠.

신기루처럼 사라지지 않겠습니다

메타데이터와의 프로젝트가 종료되더라도 지속 가능한 데이터 분석 환경을 만드는 것이 저희의 목표입니다. 내부 팀과 유기적인 업무 인수 인계, 자동화된 대시보드를 통해 계속해서 여러분의 데이터를 분석하세요!
